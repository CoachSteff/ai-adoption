# AI Context Directory

This directory contains structured context files designed to help AI systems understand and navigate the Open-Source AI Adoption Playbook repository.

## Purpose

These files follow the [SuperPrompt Framework](https://github.com/CoachSteff/superprompt) principles, providing structured, machine-readable context that enables AI systems to:

- Understand the repository's purpose and structure
- Navigate content efficiently
- Extract relevant information accurately
- Provide context-aware responses

## Files in This Directory

### 00-REPOSITORY-OVERVIEW.md
**Start here.** Provides comprehensive overview of the repository, its mission, structure, and key concepts.

### 01-STRUCTURE-GUIDE.md
Detailed directory structure and file organization guide. Use this to understand where content is located.

### 02-CONTENT-MAP.md
Comprehensive map of all content organized by topic, audience, and use case. Use this as a lookup table.

### 03-FRAMEWORKS-GLOSSARY.md
Glossary of frameworks, concepts, and terminology. Use this to understand key concepts and their relationships.

### 04-NAVIGATION-PATTERNS.md
Patterns and strategies for efficient navigation. Use this to learn how to extract information effectively.

## Usage for AI Systems

**Recommended Reading Order**:
1. `00-REPOSITORY-OVERVIEW.md` - Understand the repository
2. `01-STRUCTURE-GUIDE.md` - Understand file organization
3. `02-CONTENT-MAP.md` - Find specific content
4. `03-FRAMEWORKS-GLOSSARY.md` - Understand frameworks
5. `04-NAVIGATION-PATTERNS.md` - Learn navigation strategies

**Quick Reference**:
- Need overview? → `00-REPOSITORY-OVERVIEW.md`
- Need to find a file? → `01-STRUCTURE-GUIDE.md`
- Need content by topic? → `02-CONTENT-MAP.md`
- Need framework definition? → `03-FRAMEWORKS-GLOSSARY.md`
- Need navigation help? → `04-NAVIGATION-PATTERNS.md`

## Framework

These files follow **CoachSteff's CRAFTER (SuperPrompt Framework v0.1)** principles:

- **C**ontext - Repository environment and constraints
- **R**ole - What expertise the AI brings
- **A**ction - Step-by-step navigation tasks
- **F**ormat - Structured markdown format
- **T**arget - AI systems reading this repository
- **E**xamples - Navigation examples included
- **R**efining - Iteration guidance provided
